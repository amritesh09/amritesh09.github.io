MTP at IIT Delhi July'14 - May'15

Deleloped an Object storage system based on HDFS
The purpose of this project is to develop an Open source Object storage system that is highly, durable, scalable and reliable. Our proposed system is highly inspired from Amazon S3. We used Hadoop Distributed File System (HDFS) java API to implement our system. We propose the architecture of Object Storage system based on Hadoop. The final goal of the system is to provide REST based access to our Object Storage system that exists on top of HDFS.

Text - classification Jan'14 - June'14

Improved Naive Bayes algorithm for text classification
Automatic text classification is becoming increasingly vital nowadays. The two most common approaches for text classification using Naive Bayes Approach consider either Multivariate Bernoulli model or Multinomial model. The Multinomial model is usually observed to give better results for large number of features. We suggest some improvements in the Multivariate Bernoulli Naive Bayes Approach. We were able to achieve an average accuracy of about 99% as opposed to about 85% using basic Multivariate Naive Bayes Approach. Typing Biometrics Aug'11 - Nov'11

User Authentication Through Typing Biometrics: An Implementation
Password authentication is the most commonly used identification method used today for most computer applications. Despite potentially high usability of identification by typing patterns, this topic has received little attention in both the research and general community. In this paper I present a simple but efficient statistical detection model for constructing users’ identity from their typing patterns. The metrics used for this purpose are time to press keys, time between pressing keys and overall speed of typing. I have tried to develop an algorithm that uses all the information available from users’ typing and minimize the false rejection rate(FRR) and false acceptance rate (FAR).I have also proposed a system that provides different levels of security based on the consistency of users’ ty ping so that the system can be useful for all kinds of users.
